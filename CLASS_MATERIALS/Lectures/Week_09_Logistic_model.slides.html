Relating values to probabilities

The logistic function

\[\begin{aligned} f(x) = \frac{1}{1 + e^{-x}} . \end{aligned}\]

is the inverse of the logit function

\[\begin{aligned} f^{-1}(x) = \log\left(\frac{x}{1-x}\right) . \end{aligned}\]

Back to Stan

  1. Each player has their own batting avg, \(\theta_i\).

  2. Batting averages tend to be different for different positions.

  3. After accounting for position (\(p_i\)), \(\theta\) varies with height (\(h_i\)) and/or weight (\(w_i\)).

\[\begin{aligned} Z_i &\sim \Binom(N_i, \theta_i) \\ \theta_i &\sim \Beta(\mu_i \kappa_{p_i}, (1-\mu_i) \kappa_{p_i}) \\ \mu_i &= \logistic(\gamma_{p_i} + \alpha_h h_i + \alpha_w w_i) \\ \kappa_p &\sim \Gam(0.1, 0.1) \\ \gamma_{p} &\sim \Normal(0, 5) \\ \alpha_h &\sim \Normal(0, 5) \\ \alpha_w &\sim \Normal(0, 5) \end{aligned}\]

Questions:

  1. How much do we expect \(\gamma\) to differ for pitchers versus catchers?

  2. What would a value of \(\alpha_h = 1\) mean?

  3. What would be a range of plausible values for \(\alpha_h\) and \(\alpha_w\)?

  4. Are the priors sensible?

Note: well center and scale weight and height, so that \(h\) and \(w\) will be in units of standard deviations from the mean.

logistic_model <- "
data {
    int N;   // number of players
    int hits[N];
    int at_bats[N];
    int npos; // number of positions
    int position[N];
    vector[N] height;
    vector[N] weight;
}
parameters {
    real<lower=0, upper=1> theta[N];
    vector[npos] gamma;
    real alpha_w;
    real alpha_h;
    vector<lower=0>[npos] kappa;
}
model {
    vector[N] mu;
    mu = inv_logit(gamma[position] + alpha_h * height + alpha_w * weight);
    theta ~ beta(mu .* kappa[position], (1-mu) .* kappa[position]);
    hits ~ binomial(at_bats, theta);
    kappa ~ gamma(0.1, 0.1);
    gamma ~ normal(0, 5);
    alpha_h ~ normal(0, 5);
    alpha_w ~ normal(0, 5);
} "
